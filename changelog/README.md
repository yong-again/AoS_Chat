# 수정 이력 (Changelog)

RAG 파이프라인의 문제 수정 내역을 **날짜별 파일**로 기록합니다.

## 작성 규칙

- 파일명: `YYYY-MM-DD.md` (해당 날짜의 수정을 한 파일에 모음)
- 각 항목은 **증상 → 원인 → 수정 → 검증** 순서로 작성
- 수정한 파일명을 명시하고, 미해결 과제는 항목 끝에 남김
